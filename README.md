# protogen

Use ``proto3`` schemas to generate C++ classes which serialize and deserialize JSON messages.

## Features

Supported output programming languages:
- [x] C++98
- [ ] C99
- [ ] Java

Supported field types:
- [ ] repeated
- [ ] double
- [ ] float
- [ ] int32
- [ ] int64
- [ ] uint32
- [ ] uint64
- [ ] sint32
- [ ] sint64
- [ ] fixed32
- [ ] fixed64
- [ ] sfixed32
- [ ] sfixed64
- [ ] bool
- [ ] string
- [ ] bytes
- [ ] any
- [ ] oneof
- [ ] maps
